# URC_ARM Workspace Collection

This repository tracks both the Humble and Orangewood workspaces that make up the URC arm control stack.

- `Humble_WS` contains container tooling, optimized IK utilities, and the ROS 2 Humble development scripts.
- `orangewood_ws` is the simulation-focused workspace with the orangewood_sim_stack and supporting assets.
